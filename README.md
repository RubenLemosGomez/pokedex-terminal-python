## Uso

Si usas `pip`:

    ```bash
    python3.13 -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
    python3.13 main.py
    ```

Si usas `uv`:

    ```bash
    uv venv
    source .venv/bin/activate
    uv sync
    uv run main.py
    ```
